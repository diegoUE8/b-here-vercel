## Development
___

### Install packages
```
npm install
```
___

### Create certificates
```
npm run cert
```
___

### Build, Vendors
```
gulp build --target vendors
```
___

### Build & Watch 
```
gulp start
```
___

### Serve 
```
npm run serve
```

## Distribution
___
### Build Dist
```
gulp build --target dist
```
___
### Serve Node Dist app
```
npm run serve --dist
```
